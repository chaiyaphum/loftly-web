ข้อจำกัดที่ควรรู้

staging · ยังไม่ production — feature ใช้ได้หมดแต่มีข้อจำกัด:
  • Magic-link อีเมลยังส่งไม่จริง — Resend ยังไม่ได้ provision · login ผ่าน OAuth (Google/Apple/LINE) เท่านั้น
  • รีวิวบัตรยังน้อย — 6 บัตรใน catalog · เต็ม 40+ หลังเปิดตัวจริง
  • Sentry error tracking ยัง 🚫 — ไม่มี error ที่ส่งไป Sentry
  • Promo ticker บน landing — แสดงเฉพาะเมื่อ deal-harvester sync < 72 ชม. · ถ้าไม่เห็นแปลว่า backend sync ยังไม่ทัน
  • Valuations สำรอง 4 สกุล — KrisFlyer/Asia Miles/Bonvoy/ROP เป็น fallback data จนกว่า weekly job จะ populate ตัวจริง
